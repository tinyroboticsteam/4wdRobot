двойка - запускает вращение колес обоих бортов, обеспечивающее движение вперед с некоей начальной скоростью (100).
восьмерка - запускает вращение колес обоих бортов, обеспечивающее движение назад с некоей начальной скоростью (100).
четверка - запускает вращение колес обоих бортов в противоположных направлениях, с некоей начальной равной по модулю скоростью (100), что приводит к развороту на месте против часовой стрелки.
шестерка - запускает вращение колес обоих бортов в противоположных направлениях, с некоей начальной равной по модулю скоростью (100), что приводит к развороту на месте по часовой стрелке.

единица - прибавляет к модулю текущей скорости левого борта 40 единиц, не изменяя направление вращения. при остановленных моторах левого борта не начинает движение, при превышении значения скорости 255, задает 255.
тройка - прибавляет к модулю текущей скорости правого борта 40 единиц, не изменяя направление вращения. при остановленных моторах правого борта не начинает движение, при превышении значения скорости 255, задает 255.
семерка - отнимает от модуля текущей скорости левого борта 40 единиц, не изменяя направление вращения. при остановленных моторах левого борта не начинает движение, при достижении значения скорости менее 0, задает 0.

девятка - отнимает от модуля текущей скорости правого борта 40 единиц, не изменяя направление вращения. при остановленных моторах правого борта не начинает движение, при достижении значения скорости менее 0, задает 0.

пятерка - стоп. отменяет движение обоих бортов.
